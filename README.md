# Telugu Voice AI Agent

A voice-first, agentic AI system built in Telugu that helps users identify and apply for government welfare schemes.

## Features
- Voice input & voice output (STT → LLM → TTS)
- Native Telugu language pipeline
- Agentic workflow (Planner → Executor → Evaluator)
- Tool usage (eligibility logic, scheme retrieval)
- Conversation memory across turns
- Failure handling for incomplete or incorrect inputs

## Tech Stack
- Python
- FastAPI
- Speech-to-Text (Whisper)
- LLM APIs
- Text-to-Speech
- HTML/CSS frontend
